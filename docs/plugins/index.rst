.. _intro_plugins:

=======
Plugins
=======

These plugins cover some basis layout components commonly used.

You can disable some plugin from setting ``BLOCKS_ENABLED_PLUGINS``.

Although there is no layout stylesheet for their frontend rendering, they aims
to follow a layout direction so they have some required fields.

But finally, with respect to the required fiels, the plugin templates and their
stylesheets will be the final controller for their layout.
